1. USER AVAILABILITY
2. USER BOOKINGS
3. EVENTS
4. USER FOR HIS ACCOUNTS CREDENTIIALS AND INFO UPDATION
5. Extra packages used express middleware cors bcrypt nodemon,jwt and doteenv and mongoose
